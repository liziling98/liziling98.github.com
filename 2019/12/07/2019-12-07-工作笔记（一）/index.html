然后创建新的虚拟机，选择自定义安装（高级），选择硬件兼容性为最高，选择稍后安装虚拟系统，下拉菜单中选择Linux和Ubuntu 64位，自己定义安装位置之后，根据电脑配置选择处理器和内存，网络类型选择网络地址转换(NAT)，I/O控制器类型选择LSI Logit(L)，磁盘类型选择SCSI(S)，这些都是默认，然后磁盘选择创建新虚拟磁盘，自定义大小，确认磁盘文件后，在已准备好创建虚拟机页面选择自定义硬件，选定已经下载好的Ubuntu镜像，完成

3. Java程序打包到远程服务器运行

以我自己运行的QQRobot为例子，将Java机器人打包成jar文件，打开IDEA，在File->Project Structure->Artifacts中点击+，选择Jar->From modules with dependencies，然后输入主类为QQROBOT.QQRobot，ok后选中include in project build，完成后点击Build->build project, 在QQRobot_Java/out/ artifacts/ \QQRobot_Java_jar目录下会有生成的jar文件，在服务器中运行：nohup java -jar QQRobot_Java.jar -> out.txt &

5. coolq远程服务器运行

coolq的docker镜像为coolq/wine-coolq

其中/coolq是自己新建的目录，用来存放映射来的数据，从9000端口映射到8080端口，服务器的8080端口需要打开，VNC_PASSWD设置成8位，然后可以登录http://ip:8080，访问coolq

如果想让服务器上的coolq拥有更多功能，可以把下载的cpk插件传输到/coolq/app文件夹下，/coolq文件夹是上文命令中用户自建的文件夹

7. ubuntu常用命令

top c查看cpu占用等，shift+f可以进行更多选择
echo 1 > /proc/sys/vm/drop_caches 清理缓存
free -m 查看内存使用状态
du -h –max-depth=1 查看当前目录下子目录所占磁盘空间大小
vim下 :q!为不保存退出 :wq为保存退出
tar cvf x.tar x 用来将文件压缩成tar
tar xvf x.tar 解压tar文件
sudo rm -rf xx 强制删除xx
sudo rm xx 删除xx
crontab -l 查看定时任务
crontab -e 在vim模式下对定时任务进行修改
