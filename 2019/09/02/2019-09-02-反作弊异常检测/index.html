反作弊异常检测

发表于 2019-09-02 00:00:00 | 更新于: 2019-09-02 20:40:10 | 分类于 机器学习

作弊行为

手机黑卡，无实名认证，账号商用于帮商家刷单、刷信誉，社交平台发广告、刷粉、刷阅读量、充当网络水军、传播色情内容、进行网络诈骗等等。

手机账号商可以利用手上的资源引流（微商，诈骗等等），刷量（刷播放量，刷赞）和薅平台羊毛（平台补贴之类）。以短视频平台为例，每条变现渠道上，可能都有黑产商贩守着薅羊毛。

  1. 广告变现

    短视频平台通常会上线广告推广业务，需要推广的用户选择与平台合作后，平台可以在替其投放广告推广。

    针对短视频网站的广告，黑灰产通过大量养号，以低于平台官方价格承接推广业务后，在大量账号资源中批量发送推广广告，以巨量账号为基础，达到官方信息流的推广效果。

  2. 电商变现

    短视频巨大的流量结合电商也是互动视频平台一个有效的变现手段。电商变现模式下，有自己商品橱窗或者承接卖货的红人，就会衍生出刷量的需求，直接提高商品的曝光量和浏览量。

  3. 引流变现

    个人IP属性并不明显的部分用户，如果想成为网红，会以刷量来完成引流，这种刷量用户个人资料里绝大部分带有微信号、微博号等其他具有更明显个人IP属性的第三方平台账号，以方便后期完成变现。

  4. 直播行业

    视频网站逃不开的色情产业链，主播/大IP用户在社交网站上吸引粉丝，引流到微信或其他社交平台进行一对一服务，牟取暴利

反作弊

什么是反作弊？反作弊的界定需要结合具体的情况确定，比如产品形态、用户类型和业务模式。如在微博这种社交媒体平台上，作弊类型主要是垃圾广告、水军、僵尸粉等；在淘宝等电商平台上，作弊类型主要是刷单、机器人账号抢秒杀等；在滴滴、美团等O2O平台上，作弊类型主要是刷单行为；自媒体平台上，作弊类型主要是刷粉丝量、刷赞等等；搜索平台上，常见的作弊类型主要是垃圾链接。

如何察觉作弊行为？如果我是一名在职反作弊职员，首先会假设如果我是一名非法牟利人员，根据我对公司平台产品的了解，能够构思出哪些黑色产业盈利空间。把握这点后，可以顺藤摸瓜清理掉黑产的作弊空间。当然，随着产品的不断更新，作弊行为也会发生变化，需要时刻保持思考。

如何防止作弊？可以先思考黑产分子为什么要做黑产，庞大的利润会催生黑产产业链的发展，那么直接摆公式：利润 = 收入 - 成本。如果能够有效降低黑产利润，能在一定程度上缓解作弊行为。黑产客们的收入我们无法保证，但是尽可能地提高他们牟利地成本，压榨利润空间，平台方是可以做到的。可以同时考虑时间成本和金钱成本。

如何实际操作增加黑产商的成本？从产品的角度来讲，实名注册、登陆验证等都可以给黑卡刷量提升难度，比如谷歌验证用户是否是机器人时的图形验证码（虽然本活人用户时常无法通过机器测试，但确实能有效筛除僵尸号），交互性验证码（拖动滑块之内的，但不适合老年人和小孩使用），提升难度意味着黑产商得加钱才能攻克这些技术难关，也就压低了他们的利润空间。

当然上述的是温和派做法，让黑产商因为盈利不足而主动放弃，激进派的做法是直接进行反作弊数据收集、分析，主动出击掐断黑产行业链，比如直接排查重复IP，人工介入监控，数据平台黑名单。

异常检测

一般情况下，可以把异常检测看成是数据不平衡下的分类问题（不过如果作弊作成常态就不算是数据不平衡问题了）。异常检测问题往往是没有标签的，训练数据中并未标出哪些是异常点，因此必须使用无监督学习。不过在产品落地时，如果是一个纯粹的无监督算法，很难用一个稳定的衡量体系来评估它，也就是说它的表现怎么样很难判断，所以，模型上线后，仍然需要投入人力进行样本标注，才能对模型进行迭代优化，毕竟我们不能一拍脑袋就判断一个模型的优劣。

如何寻找异常值？比如从统计的角度来看，Z分数是一种低维特征空间中的参数异常检测方法。假定数据服从高斯分布，异常值通常在分布尾部的数据点，因此远离数据平均值超过2个或者3个标准差的值，可以视作异常值。

当然还有更直观的方法是画图，文字和数据不能直接展示出问题的时候，图像往往能帮助人们找到关键点。

结语

毕竟是面向工作学习，比较仓促。不过说到底，反作弊反的终究不是冰冷的数据，而是与人斗。
